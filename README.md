## Usage
1. To install the package, first make sure you have Rust installed.
2. Then just do:
    ```
    cargo install howami
    ```

## Contributing <!-- Using the source -->
1. Fork the repository and clone it.

### Running locally
1. `git clone https://github.com/LeptoFlare/howami.git`
2. `cd howami`
3. `cargo run`
